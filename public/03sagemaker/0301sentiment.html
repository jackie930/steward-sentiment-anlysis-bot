评论情感分析： bert-sentiment-analysis模型实验

首先下载代码

cd SageMaker
git clone https://github.com/jackie930/steward-sentiment-anlysis-bot

然后部署一个预置的endpoint

#sin
endpoint_ecr_image="204427409592.dkr.ecr.cn-northwest-1.amazonaws.com.cn/sentiment-analyisis-endpoint"

python create_endpoint.py \
--endpoint_ecr_image_path ${endpoint_ecr_image} \
--endpoint_name 'bert-sentiment-anylsis' \
--instance_type "ml.m5.xlarge"

在部署结束后，看到SageMaker控制台生成了对应的endpoint,可以使用如下客户端代码测试调用

from boto3.session import Session
import json
txt = "这瓶酸奶很好喝！"
data={"data": txt}
session = Session()
    
runtime = session.client("runtime.sagemaker")
response = runtime.invoke_endpoint(
    EndpointName='bert-sentiment-anylsis',
    ContentType="application/json",
    Body=json.dumps(data),
)

result = json.loads(response["Body"].read())
print (result)

训练部分可以参考github中readme部分。
